# 无痛省钱攒钱网站

一个帮助用户建立健康消费习惯、实现财务改善的内容型网站。

## 项目简介

本项目旨在打造一个"无痛省钱攒钱"网站，通过提供个性化省钱计划、实用技巧和情绪价值，帮助用户在不知不觉中实现财务改善。

### 核心定位

- **无痛**：不降低生活质量，不让用户变成苦行僧
- **实用**：提供可操作的省钱技巧和方案
- **个性化**：根据用户类型（年轻人/宝妈/中年人）提供针对性内容
- **情绪价值**：不说教，用故事和案例激励用户

## 功能模块

### 1. 图文文章
- 深度省钱攻略文章
- 按用户类型分类
- 支持标签筛选

### 2. 短视频
- 1-3分钟省钱技巧视频
- 快速学习实用妙招

### 3. 播客
- 省钱故事访谈
- 消费心理深度解析
- 理财书籍解读

### 4. 图书笔记
- 精选理财图书推荐
- 深度阅读笔记和核心观点提炼

### 5. 省钱计划生成器
- 根据用户输入生成个性化省钱计划
- 提供储蓄目标和执行建议

## 技术栈

- **框架**: Next.js 15 + React 19
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **部署**: 静态导出（适合GitHub Pages/Vercel等）

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000

### 构建

```bash
npm run build
```

构建后的文件在 `dist` 目录中。

## SEO 优化

本项目已内置以下 SEO 优化：

- 自动生成的 sitemap.xml
- robots.txt 配置
- 每个页面的 meta 标签
- Open Graph 标签
- 语义化 HTML 结构
